# AudioMancer

AudioMancer is a desktop audiobook production application distributed here as a packaged executable release.

## Download

Download the current Windows x64 release from the GitHub Releases page:

- [AudioMancer_Windows_x64.zip](https://github.com/DDP-Engineering/2026_AudioMancer/releases/latest/download/AudioMancer_Windows_x64.zip)

## Install

1. Download the ZIP archive.
2. Extract the full folder before starting the application.
3. Start AudioMancer with `AudioMancer.exe` or `Start-AudioMancer.cmd`.

Keep the extracted files together. The executable depends on bundled runtime files, offline help, prompt templates, settings, and license files included in the release archive.

## Distribution Scope

This public repository is used for executable release delivery and the download webpage. It does not host the application source code.
