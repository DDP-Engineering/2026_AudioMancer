Audiobook production from source text to voice-ready delivery.

AudioMancer is a Windows desktop application for building audiobook projects, preparing long-form text, organizing narrator and character voices, and producing structured audio-ready work with a local VoiceWeaver runtime.

VoiceWeaver is required for local voice generation.

AudioMancer uses VoiceWeaver as the companion runtime for voice generation. Users should download AudioMancer and all VoiceWeaver package parts from the same release.

The current public delivery is Windows-only. Linux packages are not published at this time.

Windows installation steps

  1. Download AudioMancer_Windows_x64.zip.
  2. Download all 10 VoiceWeaver .partNNN files into the same folder.
  3. Reassemble the VoiceWeaver ZIP with PowerShell.
  4. Verify the SHA256 hashes with SHA256SUMS.txt.
  5. Extract both ZIP archives before starting the applications.
  6. Start AudioMancer with AudioMancer.exe or Start-AudioMancer.cmd.

Reassemble VoiceWeaver

$parts = Get-ChildItem "VoiceWeaver-v1.2.2-windows-x64-cuda-models-3.zip.part*" | Sort-Object Name
$out = [IO.File]::Create("VoiceWeaver-v1.2.2-windows-x64-cuda-models-3.zip")
foreach ($part in $parts) {
    $in = [IO.File]::OpenRead($part.FullName)
    $in.CopyTo($out)
    $in.Dispose()
}
$out.Dispose()
Get-FileHash -Algorithm SHA256 .\VoiceWeaver-v1.2.2-windows-x64-cuda-models-3.zip
